                                Create: TwisterMill

Create: TwisterMill is an unofficial addon for the Create mod for Minecraft Forge 1.20.1.
The mod integrates wind physics from Weather2 into the Create kinetic system and adds
several custom bearing-based blocks that generate or control rotation. License: MIT

                                        Blocks:

TwisterMill Wind Bearing –-> generates RPM and Stress Units from Weather2 wind
TwisterMill Wind Directed Servo –-> vertical wind vane style bearing that aligns with wind direction using a marker
TwisterMill Servo –-> redstone controlled servo bearing for precise angle movement
TwisterMill Inverted Servo –-> alternate servo variant with persistent assembly behavior

                                        Features:

Converts wind speed into Create rotational power (RPM and SU)
Smooth wind sampling and gradual RPM ramping
Open-sky detection for the wind generator
Redstone control and comparator output
Engineer’s Goggles tooltips for diagnostics and status

                                        Configuration:

A common configuration file allows adjusting key parameters such as the SU multiplier,
wind update interval, RPM ramp update interval, ramp step size, and maximum RPM.
